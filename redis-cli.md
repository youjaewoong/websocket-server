
###기본
```
저장(key value)
set str "value1"

가져오다
get str

key 조회
keys *

key 타입확인
type str

목록 끝에 넣는다
rpush list 1
rpush list 2

keys list

키에 저장된 목록의 길이 없으면 0 목록이 아니면 error
llen list

저장된 키의 값을 개수를 인덱스를 지정하여 가져올 수있음. 음수의 경우일 경우 마지막값을 의미
-1은 마지막값, -2 마지막에서 2번째값, -3 마지막에서 3번쨰값

lrange list 0 -1
1) "1"
2) "2"
3) "3"

lrange list 1 -1
1) "2"
2) "3"
lrange list 0 -1
```
